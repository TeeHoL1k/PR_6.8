# Практическая работа 6.8.

Разработана в качестве практического задания на курсе Веб-разработчика от [Skillfactory](https://skillfactory.ru)

## Используемые технологии
* HTML
* CSS
* JavaScrip
* Web-fonts (Manrope, TildaSans, Arial)
* GitHub

## Задачи и цели
Цель проекта - закрепить знания по работе с JS и отработать на практике все возможности данного языка.

### Техническое задание
- Создать стартовую страницу с заданиями 1-5
- Создать страницу с текстом «Посмотри в консоль, там что-то важное». В консоли страницы при запуске должна отображаться информация «Привет, мир!»
- Создать событие *alert* для - *console.log, alert, prompt*
- Создать страницу с кругом, который будет менять цвет при клике - зеленый, желтый, красный
- Создать страницу с ссылкой, при клике на которую появляется окно для ввода текста. При вводе текста в окно ввода в ссылке должен меняться текст на введенный.
- Создать страницу с формой, где есть с поле ввода, кнопка отправки и текстовый блок с *id duplicateField*.
- При вводе текста в поле ввода (input) текст должен дублироваться в текстовый блок с id duplicateField.
- При нажатии кнопки «Вывести текст в консоль и удалить из формы» текст, написанный в input, должен быть выведен в консоли, а поле ввода и duplicateField должны быть очищены
- Оформить файл README с коротким описанием проекта
- Разместить проект в GitHub и создать Git Pages

Дополнительные задачи
- Использовать пользовательские стили оформления для страниц и кнопок
- Решение задачи «Светофор» модифицировано таким образом, чтобы есть три разных круга (как на светофоре), смена цвета происходит при клике на любой из них. Каждый круг может принимать только один из цветов. При этом, когда один из кругов становится «цветным», остальные становятся черными. Порядок смены цвета тот же, что и в изначальном задании
